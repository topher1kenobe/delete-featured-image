== Description ==
Automatically deletes a featured image if its associated Post is deleted.

Note: does NOT check to see if the image is associated with more than one Post.

== Installation ==
1. Upload `delete-featured-image.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
